# MarginPilot

Local-first landed-cost, contribution-profit, break-even, and reorder calculator for small e-commerce sellers.

Live site: [https://qiuxianfa.github.io/marginpilot/](https://qiuxianfa.github.io/marginpilot/)

## Run locally

```powershell
python -m http.server 4173
```

Open [http://localhost:4173](http://localhost:4173).

All calculations run in the browser. MarginPilot does not connect to a store, upload product data, or require an account.
